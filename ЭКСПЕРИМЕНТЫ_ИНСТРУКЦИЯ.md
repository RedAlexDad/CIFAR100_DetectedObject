# Инструкция по запуску экспериментов с перебором параметров

## 📊 3 эксперимента для перебора

| № | Эксперимент | Перебор | Комбинаций |
|---|-------------|---------|------------|
| 1 | **Dropout** | D1: 0.05-0.50, D2: 0.05-0.50 | 100 |
| 2 | **Weight Decay** | WD: 1e-7...3e-3 | 10 |
| 3 | **Augmentation** | 10 синхронизированных уровней | 10 |

**Итого:** 120 комбинаций

---

## 🚀 Быстрый старт

### 1. Запустить все эксперименты (120 комбинаций)

```bash
cd train_cnn
python3 run_all_experiments.py --epochs 500
```

### 2. Запустить один эксперимент

```bash
# Только Dropout (100 комбинаций)
python3 run_all_experiments.py --exp 1 --epochs 500

# Только Weight Decay (10 комбинаций)
python3 run_all_experiments.py --exp 2 --epochs 500

# Только Augmentation (10 комбинаций)
python3 run_all_experiments.py --exp 3 --epochs 500
```

### 3. Тестовый запуск (быстро)

```bash
# 5 комбинаций, 10 эпох
python3 run_all_experiments.py --exp 1 --max 5 --epochs 10
```

### 4. Проверить что будет (dry run)

```bash
python3 run_all_experiments.py --dry-run
```

## 📈 Просмотр результатов в TensorBoard

```bash
# Запуск TensorBoard
tensorboard --logdir tensorboard/

# Открыть в браузере
# http://localhost:6006
```

В TensorBoard будут доступны:

### 📊 Loss (Функция потерь):
- `Loss/train_epoch` - потери на тренировке по эпохам
- `Loss/test_epoch` - потери на тесте по эпохам
- `Loss/func_loss` - функция потерь (test loss)
- `Loss/train_batch` - потери на каждом батче (первые 5 эпох)
- `Loss/train_std`, `Loss/train_min`, `Loss/train_max` - статистика по батчам
- `Loss/test_std` - стандартное отклонение на тесте

### 🎯 Accuracy (Точность):
- `Accuracy/train_epoch` - точность на тренировке
- `Accuracy/test_epoch` - точность на тесте
- `Accuracy/train_batch` - точность на каждом батче (первые 5 эпох)
- `Best_Accuracy` - лучшая точность

### ⚙️ Optimizer (Оптимизатор):
- `Optimizer/learning_rate` - learning rate по эпохам
- `Optimizer/momentum` - momentum
- `Optimizer/weight_decay` - weight decay

### 📉 Metrics (Метрики):
- `Metrics/train_test_loss_diff` - разница train/test loss
- `Metrics/train_test_acc_diff` - разница train/test accuracy
- `Metrics/overfitting_ratio` - коэффициент переобучения

### 🎛️ HParams (Гиперпараметры):
- `HParams/label_smoothing` - сглаживание меток
- `HParams/step_size` - шаг планировщика
- `HParams/gamma` - gamma планировщика
- `HParams/dropout_1`, `HParams/dropout_2` - значения dropout

## ⚙️ Опции командной строки

| Опция | Описание | Пример |
|-------|----------|--------|
| `--exp N` | Номер эксперимента (0=все) | `--exp 1` |
| `--epochs N` | Количество эпох | `--epochs 500` |
| `--max N` | Макс. комбинаций на эксперимент | `--max 10` |
| `--tb-dir DIR` | Папка для TensorBoard | `--tb-dir results` |
| `--dry-run` | Показать план без запуска | `--dry-run` |
